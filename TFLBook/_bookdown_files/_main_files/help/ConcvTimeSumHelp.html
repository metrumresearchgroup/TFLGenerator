ConcvTimeSumR Documentation

Concentration vs time profiles for groups of patients (summary plot)

Description

Longitundinal plot showing summaries of concentration vs time profiles

Usage

ConcvTimeSum(datFile, groupBy = "DOSE", xBy = "TAFD", yBy = "DV",
  sumType = "mean", sumVar = "sd", markBy = "DOSE",
  markByType = "Discrete", preserveMarkByLevels = F, Color = T,
  xLimit = NULL, yLimit = NULL, minorTicks = FALSE, minorTickNum = 10,
  xForm = waiver(), yForm = waiver(), xScale = "identity",
  yScale = "log10", Title = "Individuals by Dose Cohort", xLab = "Time",
  yLab = "Concentration", fF = "", facetByRow = ".", facetByCol = ".",
  fnrow = NULL, fncol = NULL, fscales = "fixed", facetType = "wrap",
  themeUpdate = list(), themeTextSize = 14, themePlotTitleSize = 1.2,
  themeAxisTxtSize = 0.8, themeAxisTxtColour = "black",
  themeAxisTitleTxtSize = 0.9, themeAxisTitleColour = "black",
  themePanelBackgroundFill = "white", themePanelGridSize = NULL,
  themePanelGridColour = "white", themePanelLineType = 1,
  themePanelTitleSize = 1.2, themePlotTitleColour = "black",
  themePlotLegendPosition = "right", srcAdd = TRUE, srcPath = ".",
  srcName = "script", figPath = "../deliv/figure", figName = "Rplot.pdf",
  ...)

Arguments

datFile

A data.frame containing the necessary variables for the figure

groupBy

Character, name of the column of datFile on which to group the summaries by

xBy

Name of the variable denoting the x axis (typically time) in datFile

yBy

Ignored

sumType

The summarized response variable in datFile; this function assumes the summary has already been performed

sumVar

Ignored

markBy

Character, name of the column of datFile on which to add a color scale

markByType

"Discrete" or "Continuous" for the type of color scale to use on markBy

preserveMarkByLevels

Logical, matches the call to manipDat that generated datFile when using the GUI. Can be ignored when running outside of the GUI

Color

Logical, colored or greyscale?

xLimit

Two element numeric vector giving the lower and upper limits of xBy

yLimit

Two element numeric vector giving the lower and upper limits of the concentration axis (y)

xForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the x-axis

yForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the y-axis

xScale

One of "identity", "log", or "log-10"

yScale

One of "identity", "log", or "log-10"

Title

A title for the plot

xLab

Label for the x axis

yLab

Label for the y axis

fF

A character string acting as a key for re-labeling the facet values. Takes the form "value1,value2;label1,label2"

fnrow

Number of rows in faceted layout

fncol

Number of columns in faceted layout

fscales

"fixed", "free", "free_y", or "free_x" as seen in the argument to facet_wrap

facetBy

Column in datFile on which to facet by

Title

String, the title for the plot

theme*

Various controls for the ggplot2 theme

Details

This function relies upon the user (or the TFL generator app) to provide sumamrized data in datFile. The TFL generator itself uses the manipDat function to perform these summaries, for example.

Value

A concentration vs time ggplot2 object of class TFL which can be plotted or printed with the corresponding method.

Examples

data("twoCmt")
twoCmt=twoCmt%>%mutate_each(funs(factor),DOSE,SEX)
ConcvTimeSum(datFile = manipDat(twoCmt,groupBy = 'SEX',sumThis = T))
