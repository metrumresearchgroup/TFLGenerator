QQplotR Documentation

Create a standard Quantile-Quantile plot

Description

Creates a standard quantile-quantile plot using a qnorm estimation of the data.

Usage

QQplot(datFile, xBy = "ETA1", groupBy = NULL, markBy = NULL, xLimit = NULL, yLimit = NULL, xForm = waiver(), yForm = waiver(), xScale = "identity", yScale = "identity", Title = "", xLab = "sprintf(\'Theoretical %s\', xBy)", yLab = "sprintf(\'Observed %s\', xBy)", facetBy = "", ...)

Arguments

datFile

data.frame output of a NM run containing ER-SSAP standard column names and numeric-only columns

xBy

character column name for sample column (quantiles will be created using the qnorm estimation within stat_qq)

groupBy

character colum name to group data within the plot

markBy

character column name for characteristic groupings within the plot (such as color and shape)

xLimit

vector input for x-axis limits example: c(0,10)

yLimit

vector input for y-axis limits example: c(.1,100000)

xForm

function format of the x-axis (options: comma, scientific, percent or waiver() for no formatting)

yForm

function format of the y-axis (options: comma, scientific, percent or waiver() for no formatting)

xScale

function format for the x-axis (options, "identity", log10_trans(), log_trans())

yScale

function format for the y-axis (options, "identity", log10_trans(), log_trans())

Title

character figure title

xLab

character x-axis title

yLab

character y-axis title

facetBy

character column name for optional faceting

Details

Standard post-analysis qq plot.

Value

returns ggplot object (grob) that must be saved or printed

Examples

data("twoCmt")
ex1=QQplot(twoCmt)
ex2=QQplot(twoCmt,markBy = 'SEX')
ex3=QQplot(twoCmt,facetBy = 'DOSE')
list(ex1,ex2,ex3)
