OBSvIPREDR Documentation

Standard goodness of fit plots for observed versus predicted values

Description

Yields a ggplot object comparing observed values to predicted values

Usage

OBSvIPRED(datFile, xBy = "IPRE", yBy = "DV", groupBy = NULL,
  markBy = NULL, markByType = "Discrete", obsLabel = NULL,
  smoothType = ifelse(nrow(datFile) <= 1000, "loess", "gam"), xLimit = "",
  xBreaks = waiver(), xForm = waiver(), xScale = scales::log10_trans(),
  Title = "", xLab = "Individiual Prediction",
  yLab = "Observed Plasma Concentration", facetBy = "", fF = "",
  fnrow = "", fncol = "", fscales = "fixed", minorTicks = NULL,
  minorTickNum = 10, themeUpdate = list(), themeTextSize = 14,
  themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", themePlotLegendPosition = "right", ...)

Arguments

datFile

data.frame, contains the necessary variables to produce the output

xBy

character, Column name for X-axis, Default: 'IPRE'

yBy

character, Column name for Y-axis, Default: 'DV'

groupBy

character, column name for groupings within the output, Default: NULL

markBy

character, column name on which to mark by different colors, Default: NULL

markByType

character, controls the type of variable expected mapped to markBy c("Discrete", "Continuous"), Default: 'Discrete'

obsLabel

character, if set to a column name annotation of text is used instead of points, Default: NULL

smoothType

function|character, function used for interpolation, Default: ifelse(nrow(datFile) <= 1000,"loess","gam")

xLimit

numeric, Two element vector giving the lower and upper limits of the X-axis, Default: ”

xBreaks

numeric, Vector input for X-axis breaks, Default: waiver()

xForm

function|character, Format of the X-axis variable tick label, Default: waiver()

xScale

function|character, Scale transformtion for the X-axis variable, Default: scales::log10_trans()

Title

character, Figure title, Default: ”

xLab

character, Label of X-axis, Default: 'Individiual Prediction'

yLab

character, Label of Y-axis, Default: 'Observed Plasma Concentration'

facetBy

character, column name for figure faceting, Default: ”

fF

character, Labels for the facet levels in the form "Level1,Label1;Level2,Label2", Default: ”

fnrow

integer, Number of rows passed to facet_wrap, Default: ”

fncol

integer, Number of columns passed to facet_wrap, Default: ”

fscales

character, Scale type for facet_wrap c("fixed","free","free_y","free_x"), Default: 'fixed'

minorTicks

character, sides to add minorticks, t,b,l,r; if NULL (default) no minor ticks are created, Default: NULL

minorTickNum

integer, number of minor ticks between major ticks, Default: 10

themeUpdate

list, with theme elements to replace session theme, Default: list()

themeTextSize

numeric, overall plot text size, Default: 14

themePlotTitleSize

numeric, plot title text size relative to themeTextSize, Default: 1.2

themeAxisTxtSize

numeric, plot axis text size relative to themeTextSize, Default: 0.8

themeAxisTxtColour

character, axis text colour, Default: 'black'

themeAxisTitleTxtSize

numeric, plot axis title text size relative to themeTextSize, Default: 0.9

themeAxisTitleColour

character, axis title text colour, Default: 'black'

themePanelBackgroundFill

character, plot background colour, Default: 'white'

themePanelGridSize

numeric, grid line size, Default: NULL

themePanelGridColour

character, grid lines colour, Default: 'white'

themePanelLineType

numeric, grid line type, Default: 1

themePanelTitleSize

numeric, panel title text size relative to themeTextSize, Default: 1.2

themePlotTitleColour

character, panel title text colour, Default: 'black'

themePlotLegendPosition

character, legend position, Default: 'right'

...

Details

Standard goodness of fit showing DV versus population prediction or individual prediction.

Value

returns ggplot object (grob) that must be saved or printed

Examples

data("twoCmt")
OBSvIPRED(twoCmt%>%dplyr::filter(DV>0),xBy='IPRED')
OBSvIPRED(twoCmt,xBy='IPRED',xScale = 'identity',markBy = 'SEX')
