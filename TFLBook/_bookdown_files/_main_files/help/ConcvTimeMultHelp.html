ConcvTimeMultR Documentation

Multipanel concentration vs time profiles for individual patients

Description

Longitundinal plot showing individual concentration vs time profile, including observed, conditional, and unconditional (on random effects) values of concentration

Usage

ConcvTimeMult(datFile, groupBy = "NMID", xBy = "TAFD", yBy = "DV",
  ipredVar = "IPRE", predVar = "PRED", doseVar = "DOSE",
  doseLab = "mg/kg", markBy = "DOSE", preserveMarkByLevels = FALSE,
  Color = TRUE, rugVar = NULL, rugVals = 1, rugCols = "red",
  blqVal = NULL, xLimit = NULL, yLimit = NULL, xForm = waiver(),
  yForm = waiver(), xScale = "identity", yScale = "log10",
  Title = "Individuals", xLab = "Time", yLab = "Concentration",
  fncol = 3, fnrow = 3, ID = NULL, minorTicks = NULL,
  minorTickNum = 10, genAll = TRUE, page = 1, themeUpdate = list(),
  themeTextSize = 14, themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", srcAdd = TRUE, srcPath = ".",
  srcName = "script", figPath = "../deliv/figure", figName = "Rplot.pdf",
  ...)

Arguments

datFile

data.frame, contains the necessary variables to produce the output

groupBy

character, column name for groupings within the output, Default: 'NMID'

xBy

character, Column name for x-axis, Default: 'TAFD'

yBy

character, Column name for Y-axis, Default: 'DV'

ipredVar

character, Column name of the individual prediction variable, Default: 'IPRE'

predVar

character, Column name containing the conditional prediction, Default: 'PRED'

doseVar

character, Column name of dosing variable , Default: 'DOSE'

doseLab

character, Column name of labels for the dosing variable, Default: 'mg/kg'

markBy

character, column name on which to mark by different colors, Default: 'DOSE'

preserveMarkByLevels

boolean, to preserve the levels of the markBy variable, Default: FALSE

Color

boolean, controls if the plot has color aesthetic, Default: TRUE

rugVar

character, Column name of variable to create rug layer (Default NULL), Default: NULL

rugVals

numeric, vector of values rugVar is displayed, Default: 1

rugCols

character, Color of rug ticks, Default: 'red'

blqVal

numeric, defines the 'Below Level of Quantification (BLQ)' horizontal reference line, DEFAULT: NULL, Default: NULL

xLimit

numeric, Two element vector giving the lower and upper limits of the x-axis, Default: NULL

yLimit

numeric, Two element vector giving the lower and upper limits of the y-axis, Default: NULL

xForm

function|character, Format of the x-axis variable tick label, Default: waiver()

yForm

function|character, Format of the y-axis variable tick label, Default: waiver()

xScale

function|character, Scale transformtion for the x-axis variable, Default: 'identity'

yScale

function|character, Scale transformtion for the y-axis variable, Default: 'log10'

Title

character, Figure title, Default: 'Individuals'

xLab

character, Label of X-axis, Default: 'Time'

yLab

character, Label of Y-axis, Default: 'Concentration'

fncol

integer, Number of columns passed to facet_wrap, Default: 3

fnrow

integer, Number of rows passed to facet_wrap, Default: 3

ID

numeric, vector of the patient IDs to plot, Default: NULL

minorTicks

character, sides to add minorticks, t,b,l,r; if NULL (default) no minor ticks are created, Default: NULL

minorTickNum

integer, number of minor ticks between major ticks, Default: 10

genAll

boolean, generate all pages Default: TRUE

page

numeric, vector of pages to output Default: 1

themeUpdate

list, with theme elements to replace session theme, Default: list()

themeTextSize

numeric, overall plot text size, Default: 14

themePlotTitleSize

numeric, plot title text size relative to themeTextSize, Default: 1.2

themeAxisTxtSize

numeric, plot axis text size relative to themeTextSize, Default: 0.8

themeAxisTxtColour

character, axis text colour, Default: 'black'

themeAxisTitleTxtSize

numeric, plot axis title text size relative to themeTextSize, Default: 0.9

themeAxisTitleColour

character, axis title text colour, Default: 'black'

themePanelBackgroundFill

character, plot background colour, Default: 'white'

themePanelGridSize

numeric, grid line size, Default: NULL

themePanelGridColour

character, grid lines colour, Default: 'white'

themePanelLineType

numeric, grid line type, Default: 1

themePanelTitleSize

numeric, panel title text size relative to themeTextSize, Default: 1.2

themePlotTitleColour

character, panel title text colour, Default: 'black'

srcAdd

boolean, add source caption to output, Default: TRUE

srcPath

character, source script path, Default: '.'

srcName

character, source script name, Default: 'script'

figPath

character, Output figure path, Default: '../deliv/figure'

figName

character, Output figure name, Default: 'Rplot.pdf'

...

Value

A concentration vs time ggplot2 object of class TFL which can be plotted or printed with the corresponding method.

See Also

addlExpand

Examples

data("twoCmt")
twoCmt=twoCmt%>%dplyr::mutate(DOSE=factor(DOSE))
## Not run: ConcvTimeMult(datFile = twoCmt,ipredVar='IPRED')
ConcvTimeMult(datFile = twoCmt,ipredVar='IPRED',genAll=FALSE,page=1)
ConcvTimeMult(datFile = twoCmt,ipredVar='IPRED',genAll=FALSE,page=2,fnrow =5,fncol=4)

rug.data=addlExpand( tab =file.path(find.package('TFL'),'external/mi210/510/510.tab'),
demog = file.path(find.package('TFL'),'external/mi210/poppk_wcovs.csv'),
group.vars = c('ID','EVID','AMT'),
demog.fillVars = c('HT','WT','CLCR','SEX','AGE'),
demog.keepVars = c('RATE')
)

ConcvTimeMult(datFile = rug.data,
              groupBy='ID',
              xBy = 'TIME',
              ipredVar = 'IPRED',
              genAll = FALSE,
              ID=1:9,
              rugVar = 'EVID',
              rugVals = 1)
