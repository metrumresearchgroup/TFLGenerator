ConcvTimeMultR Documentation

Multipanel concentration vs time profiles for individual patients

Description

Longitundinal plot showing individual concentration vs time profile, including observed, conditional, and unconditional (on random effects) values of concentration

Usage

ConcvTimeMult(datFile, groupBy = "NMID", xBy = "TAFD", yBy = "DV",
  ipredVar = "IPRE", predVar = "PRED", doseVar = "DOSE",
  doseLab = "mg/kg", markBy = "DOSE", preserveMarkByLevels = F,
  Color = T, rugVar = NULL, rugVals = 1, rugCols = "red",
  blqVal = NULL, xLimit = NULL, yLimit = NULL, xForm = waiver(),
  yForm = waiver(), xScale = "identity", yScale = "log10",
  Title = "Individuals", xLab = "Time", yLab = "Concentration",
  ncol = 3, nrow = 3, ID = NULL, minorTicks = FALSE,
  minorTickNum = 10, genAll = T, page = 1, themeUpdate = list(),
  themeTextSize = 14, themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", srcAdd = TRUE, srcPath = ".",
  srcName = "script", figPath = "../deliv/figure", figName = "Rplot.pdf",
  ...)

Arguments

datFile

A data.frame containing the necessary variables for the figure

groupBy

Column name that identifies the subset of datFile to be used in each of the panels; typically, this is "NMID" or "ID.

xBy

Name of the variable denoting the x axis (typically time) in datFile

yBy

Name of the variable containing the observed concentration values

ipredVar

Individual prediction column

predVar

Population prediction column

doseVar

Column name of datFile in which dosing labels are held. If specified, these doses will appear as titles per patient

doseLab

A label to be appended to the value in doseVar

markBy

ignored

rugVar

Column name of variable to create rug layer (default NULL)

rugVals

Numeric vector of values rugVar is displayed (default c(1))

rugCols

Character value that defines color of rug ticks (default 'red')

blqVal

Numeric value that defines the 'Below Level of Quantification (BLQ)' horizontal reference line (default NULL)

xLimit

Two element numeric vector giving the lower and upper limits of xBy

yLimit

Two element numeric vector giving the lower and upper limits of the concentration axis (y)

Title

String, the title for the plot

ID

Numeric vector, the ID's in datFile[,idVar] you wish to plot

minorTicks

Logical, include minor ticks?

minorTickNum

Integer, number of minor ticks between major ticks

theme*

Various controls for the ggplot2 theme

Value

A concentration vs time ggplot2 object of class TFL which can be plotted or printed with the corresponding method.

Examples

data("twoCmt")
twoCmt=twoCmt%>%mutate(DOSE=factor(DOSE))
## Not run: ConcvTimeMult(datFile = twoCmt,ipredVar='IPRED')
ConcvTimeMult(datFile = twoCmt,ipredVar='IPRED',genAll=F,page=1)
ConcvTimeMult(datFile = twoCmt,ipredVar='IPRED',genAll=F,page=2,nrow =5,ncol=4)
