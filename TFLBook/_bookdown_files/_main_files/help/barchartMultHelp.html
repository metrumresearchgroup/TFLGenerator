barchartMultR Documentation

Bar charts

Description

barchartMult returns a barchart for each categorical variable listed.

Usage

barchartMult(datFile, ID = "NMID", catList = list(c("SEX", "Gender"),
  c("RACE", "Race"), c("DIS", "Disease status")), themeUpdate = list(),
  themeTextSize = 14, themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", fnrow = 1, srcAdd = TRUE,
  srcPath = ".", srcName = "script", figPath = "../deliv/figure",
  figName = "Rplot.pdf", ...)

Arguments

datFile

data.frame, contains the necessary variables to produce the output

ID

numeric, vector of the patient IDs to plot, Default: 'NMID'

catList

list, list of two element vectors giving the column name and rename, Default: list(c("SEX", "Gender"), c("RACE", "Race"), c("DIS", "Disease status"))

themeUpdate

list, with theme elements to replace session theme, Default: list()

themeTextSize

numeric, overall plot text size, Default: 14

themePlotTitleSize

numeric, plot title text size relative to themeTextSize, Default: 1.2

themeAxisTxtSize

numeric, plot axis text size relative to themeTextSize, Default: 0.8

themeAxisTxtColour

character, axis text colour, Default: 'black'

themeAxisTitleTxtSize

numeric, plot axis title text size relative to themeTextSize, Default: 0.9

themeAxisTitleColour

character, axis title text colour, Default: 'black'

themePanelBackgroundFill

character, plot background colour, Default: 'white'

themePanelGridSize

numeric, grid line size, Default: NULL

themePanelGridColour

character, grid lines colour, Default: 'white'

themePanelLineType

numeric, grid line type, Default: 1

themePanelTitleSize

numeric, panel title text size relative to themeTextSize, Default: 1.2

themePlotTitleColour

character, panel title text colour, Default: 'black'

fnrow

integer, Number of rows passed to facet_wrap, Default: 1

srcAdd

boolean, add source caption to output Default: TRUE, Default: TRUE

srcPath

character, source script path, Default: '.'

srcName

character, source script name, Default: 'script'

figPath

character, Output figure path, Default: '../deliv/figure'

figName

character, Output figure name, Default: 'Rplot.pdf'

...

Value

A barchart ggplot2 object of class TFL which can be plotted or printed with the corresponding method.

Examples

data("twoCmt")
barchartMult(datFile = twoCmt)
barchartMult(datFile = twoCmt,catList = list(c('SEX','Gender')))
barchartMult(datFile = twoCmt,fnrow=2)
