covConR Documentation

Scatterplots

Description

Scatterplots comparing (multiple) continuous variables

Usage

covCon(datFile, conList_x, conList_y, plotCols = 2, xLimit = NULL,
  yLimit = NULL, xForm = waiver(), yForm = waiver(),
  xScale = "identity", yScale = "identity", facetBy = "", fF = "",
  fnrow = NULL, fncol = NULL, fscales = "fixed", minorTicks = NULL,
  minorTickNum = 10, corstat = NULL, themeUpdate = list(),
  themeTextSize = 14, themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", srcAdd = TRUE, srcPath = ".",
  srcName = "script", figPath = "../deliv/figure", figName = "Rplot.pdf",
  ...)

Arguments

datFile

data.frame, contains the necessary variables to produce the output

conList_x

list, list of two element vectors giving the column name and rename (x-axis)

conList_y

list, list of two element vectors giving the column name and rename (y-axis)

plotCols

integer, Number of columns to arrange plots into grid output, Default: 2

xLimit

numeric, Two element vector giving the lower and upper limits of the x-axis, Default: NULL

yLimit

numeric, Two element vector giving the lower and upper limits of the y-axis, Default: NULL

xForm

function|character, Format of the x-axis variable tick label, Default: waiver()

yForm

function|character, Format of the y-axis variable tick label, Default: waiver()

xScale

function|character, Scale transformtion for the x-axis variable, Default: 'identity'

yScale

function|character, Scale transformtion for the y-axis variable, Default: 'identity'

facetBy

character, column name for figure faceting, Default: ”

fF

character, Labels for the facet levels in the form "Level1,Label1;Level2,Label2", Default: ”

fnrow

integer, Number of rows passed to facet_wrap, Default: NULL

fncol

integer, Number of columns passed to facet_wrap, Default: NULL

fscales

character, Scale type for facet_wrap c("fixed","free","free_y","free_x"), Default: 'fixed'

minorTicks

character, sides to add minorticks, t,b,l,r; if NULL (default) no minor ticks are created, Default: NULL

minorTickNum

integer, number of minor ticks between major ticks, Default: 10

corstat

character, "R2" for adjusted R^2 or "r" for Pearson correlation, Default: NULL

themeUpdate

list, with theme elements to replace session theme, Default: list()

themeTextSize

numeric, overall plot text size, Default: 14

themePlotTitleSize

numeric, plot title text size relative to themeTextSize, Default: 1.2

themeAxisTxtSize

numeric, plot axis text size relative to themeTextSize, Default: 0.8

themeAxisTxtColour

character, axis text colour, Default: 'black'

themeAxisTitleTxtSize

numeric, plot axis title text size relative to themeTextSize, Default: 0.9

themeAxisTitleColour

character, axis title text colour, Default: 'black'

themePanelBackgroundFill

character, plot background colour, Default: 'white'

themePanelGridSize

numeric, grid line size, Default: NULL

themePanelGridColour

character, grid lines colour, Default: 'white'

themePanelLineType

numeric, grid line type, Default: 1

themePanelTitleSize

numeric, panel title text size relative to themeTextSize, Default: 1.2

themePlotTitleColour

character, panel title text colour, Default: 'black'

srcAdd

boolean, add source caption to output, Default: TRUE

srcPath

character, source script path, Default: '.'

srcName

character, source script name, Default: 'script'

figPath

character, Output figure path, Default: '../deliv/figure'

figName

character, Output figure name, Default: 'Rplot.pdf'

...

Details

In addition to reporting either the correlation coefficient or adjusted R^2, the significance of the coefficient in a linear regression of y on x is reported as well (i.e., testing whether the coefficient is non-zero). All observations are shown on the plot as well as both a linear fit and loess smooth to the data.

Value

A "TFL" object consisting of a list of plots and the intended layout for their display. The figure can be rendered with the print or plot methods.

Examples

data("twoCmt")
covCon(twoCmt,conList_x = 'IPRED',conList_y = 'ETA1',plotCols=1)
covCon(twoCmt,
conList_x = list(
list('IPRED','Individual Prediction'),
list('IRES','Individual Residual')
),
conList_y = list(
list('ETA1',expression(eta[1]))
)
)
