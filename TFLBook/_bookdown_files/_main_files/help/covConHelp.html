covConR Documentation

Scatterplots

Description

Scatterplots comparing (multiple) continuous variables

Usage

covCon(datFile, conList_x, conList_y, nCols = 2, xLimit = NULL,
  yLimit = NULL, xForm = waiver(), yForm = waiver(),
  xScale = "identity", yScale = "identity", facetBy = "", fF = "",
  fnrow = NULL, fncol = NULL, fscales = "fixed", minorTicks = FALSE,
  minorTickNum = 10, corstat = NULL, themeUpdate = list(),
  themeTextSize = 14, themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", srcAdd = TRUE, srcPath = ".",
  srcName = "script", figPath = "../deliv/figure", figName = "Rplot.pdf",
  ...)

Arguments

datFile

A data.frame containing the necessary variables for the figure

conList_x

A list of two character vectors.0 The first is a vector of the continuous variables in datFile to be plotted on the x axis, the second is a list of matching length containing "pretty" labels for each of the continuous variables

conList_y

A list of two character vectors. The first is a vector of the continuous variables in datFile to be plotted on the y axis, the second object is a list of matching length containing "pretty" labels for each of those variables

nCols

Integer, how many columns to arrange the plots into

xLimit

Two element numeric vector giving the lower and upper limits of the x axis, recycled for all x axes if multiple

yLimit

Two element numeric vector giving the lower and upper limits of the y axis, recycled for all y axes if multiple

xForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the axis

yForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the axis

xScale

One of "identity", "log", "log10"; passed to scale_x_continuous

yScale

One of "identity", "log", "log10"; passed to scale_y_continuous

facetBy

Variable name in datFile on which to facet_wrap by

fF

Labels for the facet levels in the form "Level1,Label1;Level2,Label2"

fnrow

Number of rows passed to facet_wrap

fncol

Number of columns passed to facet_wrap

fscales

Scale type for facet_wrap: "fixed", "free", "free_y", or "free_x"

minorTicks

Logical, include minor ticks?

minorTickNum

Integer, number of minor ticks between major ticks

corstat

"R2" for adjusted R^2 or "r" for Pearson correlation

theme*

Various controls for the ggplot2 theme

Details

In addition to reporting either the correlation coefficient or adjusted R^2, the significance of the coefficient in a linear regression of y on x is reported as well (i.e., testing whether the coefficient is non-zero). All observations are shown on the plot as well as both a linear fit and loess smooth to the data.

Value

A "TFL" object consisting of a list of plots and the intended layout for their display. The figure can be rendered with the print or plot methods.

Examples

data("twoCmt")
covCon(twoCmt,conList_x = 'IPRED',conList_y = 'ETA1',nCol=1)
covCon(twoCmt,
conList_x = list(
list('IPRED','Individual Prediction'),
list('IRES','Individual Residual')
),
conList_y = list(
list('ETA1',expression(eta[1]))
)
)
