covCatR Documentation

Boxplots

Description

Boxplot comparing (multiple) categorical variables to (multiple) continuous variables

Usage

covCat(datFile, catList, conList, nCols = 2, idVar = "NMID",
  yLimit = NULL, yForm = waiver(), yScale = "identity", facetBy = "",
  notches = TRUE, minorTicks = FALSE, minorTickNum = 10,
  themeUpdate = list(), themeTextSize = 14, themePlotTitleSize = 1.2,
  themeAxisTxtSize = 0.8, themeAxisTxtColour = "black",
  themeAxisTitleTxtSize = 0.9, themeAxisTitleColour = "black",
  themePanelBackgroundFill = "white", themePanelGridSize = NULL,
  themePanelGridColour = "white", themePanelLineType = 1,
  themePanelTitleSize = 1.2, themePlotTitleColour = "black", ...)

Arguments

datFile

A data.frame containing the necessary variables for the figure

catList

A list of two objects. The first object is a vector of the categorical variables in datFile to be plotted, the second object is a list of matching length containing "pretty" labels for each of the categorical variables

conList

A list of two objects. The first object is a vector of the continuous variables in datFile to be plotted, the second object is a list of matching length containing "pretty" labels for each of the continuous variables

nCols

Integer, how many columns to arrange the plots into

idVar

Column name for ID, used to report the number of patients at each level of the categorical variables

yLimit

Two element numeric vector giving the lower and upper limits of the y axis, recycled for all continuous variables if multiple

yForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the y-axis

yScale

One of "identity", "log", "log10"; passed to scale_[xy]_continuous

facetBy

Variable name in datFile on which to facet_wrap by

notches

Logical, should the boxplots be notched or not (see boxplot.stats for details)

minorTicks

Logical, include minor ticks?

minorTickNum

Integer, number of minor ticks between major ticks

Title

String, the title for the plot

theme*

Various controls for the ggplot2 theme

Details

Most calculations are passed through to boxplot.stats with a few important exceptions. Each plot includes the adjusted R squared value and p-value associated with an analysis of variance of the continuous variable on the categorical variable. Additionally, the inner 20 of the continuous variable is shown as a shaded box overlayed on the traditional boxplot.

Value

A "TFL" object consisting of a list of plots and the intended layout for their display. The figure can be rendered with the print or plot methods.

Examples

data("twoCmt")
ex1=covCat(twoCmt,catList = list('SEX'),conList = 'IPRED',nCols=1)
ex2=covCat(twoCmt,catList = list(list('SEX','Gender'),list('DOSE')),conList = list(list('IPRED'),list('ETA1',expression(eta[1]))))
list(ex1,ex2)
