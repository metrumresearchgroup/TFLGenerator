paramDistR Documentation

Parameter Distribution Plots

Description

Produces a single parameter distribution plot with density and normal distrubution overlays

Usage

paramDist(datFile, xCols = "ETA1", binWidth = 0.1, groupBy = NULL,
  xLimit = NULL, yLimit = NULL, xForm = ggplot2::waiver(),
  yForm = ggplot2::waiver(), xScale = "identity", yScale = "identity",
  Title = "", xLab = "xCols", yLab = "Percent", facetBy = "", fF = "",
  fnrow = "", fncol = "", fscales = "fixed", runno = "",
  project = getwd(), plotCols = 1, plotRows = 1, minorTicks = "bl",
  minorTickNum = 10, themeUpdate = list(), themeTextSize = 14,
  themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", ...)

Arguments

datFile

data.frame, contains the necessary variables to produce the output

xCols

character, Column name(s) for X-axis variable(s), Default: 'ETA1'

binWidth

numeric, histogram bin width, Default: 0.1

groupBy

character, column name for groupings within the output, Default: NULL

xLimit

numeric, Two element vector giving the lower and upper limits of the X-axis, Default: NULL

yLimit

numeric, Two element vector giving the lower and upper limits of the Y-axis, Default: NULL

xForm

function|character, Format of the X-axis variable tick label, Default: ggplot2::waiver()

yForm

function|character, Format of the Y-axis variable tick label, Default: ggplot2::waiver()

xScale

function|character, Scale transformtion for the X-axis variable, Default: 'identity'

yScale

function|character, Scale transformtion for the Y-axis variable, Default: 'identity'

Title

character, Figure title, Default: ”

xLab

character, Label of X-axis, Default: 'xCols'

yLab

character, Label of Y-axis, Default: 'Percent'

facetBy

character, column name for figure faceting, Default: ”

fF

character, Labels for the facet levels in the form "Level1,Label1;Level2,Label2", Default: ”

fnrow

integer, Number of rows passed to facet_wrap, Default: ”

fncol

integer, Number of columns passed to facet_wrap, Default: ”

fscales

character, Scale type for facet_wrap c("fixed","free","free_y","free_x"), Default: 'fixed'

runno

character, Vector of run NONMEM numbers, Default: ”

project

character, Home directory of the NONMEM Run, Default: getwd()

plotCols

integer, Number of columns to arrange plots into grid output, Default: 1

plotRows

integer, Number of rows to arrange plots into grid output, Default: 1

minorTicks

character, sides to add minorticks, t,b,l,r; if NULL (default) no minor ticks are created, Default: 'bl'

minorTickNum

integer, number of minor ticks between major ticks, Default: 10

themeUpdate

list, with theme elements to replace session theme, Default: list()

themeTextSize

numeric, overall plot text size, Default: 14

themePlotTitleSize

numeric, plot title text size relative to themeTextSize, Default: 1.2

themeAxisTxtSize

numeric, plot axis text size relative to themeTextSize, Default: 0.8

themeAxisTxtColour

character, axis text colour, Default: 'black'

themeAxisTitleTxtSize

numeric, plot axis title text size relative to themeTextSize, Default: 0.9

themeAxisTitleColour

character, axis title text colour, Default: 'black'

themePanelBackgroundFill

character, plot background colour, Default: 'white'

themePanelGridSize

numeric, grid line size, Default: NULL

themePanelGridColour

character, grid lines colour, Default: 'white'

themePanelLineType

numeric, grid line type, Default: 1

themePanelTitleSize

numeric, panel title text size relative to themeTextSize, Default: 1.2

themePlotTitleColour

character, panel title text colour, Default: 'black'

...

Details

Standard Eta distribution histogram plot for showing density and normal distribution of ETA

Value

returns ggplot object (grob) that must be saved or printed

Examples

data("twoCmt")
paramDist(twoCmt)
paramDist(twoCmt,xCols=c('ETA1','ETA2'),xLab=c(expression(eta[1]),expression(eta[2])))
