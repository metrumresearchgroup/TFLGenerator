ggpairsR Documentation

Correlation Comparison plot

Description

Creates a grid ggplot object with R-squared and p-values indicated in corresponding upper trianges

Usage

ggpairs(datFile, xCols = grep("ETA", names(datFile), value = TRUE), xLimit = NULL, yLimit = NULL, xForm = waiver(), yForm = waiver(), xScale = "identity", yScale = "identity", facetBy = "", ...)

Arguments

datFile

data.frame output of a NM run containing ER-SSAP standard column names and numeric-only columns

xCols

character column names

xLimit

vector input for x-axis limits example: c(0,10) of lower triangle plots

yLimit

vector input for y-axis limits example: c(.1,100000) of lower triangle plots

xForm

function format of the x-axis (options: comma, scientific, percent or waiver() for no formatting) of lower triangle plots

yForm

function format of the y-axis (options: comma, scientific, percent or waiver() for no formatting) of lower triangle plots

xScale

function format for the x-axis (options, "identity", log10_trans(), log_trans()) of lower triangle plots

yScale

function format for the y-axis (options, "identity", log10_trans(), log_trans()) of lower triangle plots

facetBy

character column name for optional faceting

Details

Lower triangle shows comparison plots of diagonal values with line of best fit, upper triangle shows R squared and p values for these comparisons

Value

returns ggplot object (grob) that must be saved or printed

Examples

data("twoCmt")
ggpairs(twoCmt,xCols=c('ETA1','ETA2'))
ggpairs(twoCmt,xCols=c('ETA1','ETA2','ETA5'))
