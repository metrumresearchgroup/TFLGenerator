distMultR Documentation

Multipanel distribution

Usage

distMult(datFile, idVar = "NMID", conList = list(c("WGTB", "Weight (kg)"),
  c("AGE", "Age (years)"), c("CL", "Creatinine Clearance (mL/min)")),
  notches = T, order = c("Density", "QQ", "Boxplot"),
  themeUpdate = list(), themeTextSize = 14, themePlotTitleSize = 1.2,
  themeAxisTxtSize = 0.8, themeAxisTxtColour = "black",
  themeAxisTitleTxtSize = 0.9, themeAxisTitleColour = "black",
  themePanelBackgroundFill = "white", themePanelGridSize = NULL,
  themePanelGridColour = "white", themePanelLineType = 1,
  themePanelTitleSize = 1.2, themePlotTitleColour = "black",
  srcAdd = TRUE, srcPath = ".", srcName = "script",
  figPath = "../deliv/figure", figName = "Rplot.pdf", ...)

Arguments

datFile

data.frame

idVar

ID variable in datFile

conList

list of two element vectors giving the continuous variables in datFile you wish to summarize, and the "pretty" names of those variables for printing

notches

Notched boxplots?

order

Some permutation of "Density", "QQ", and "Boxplot"

theme*

various parameters for the plot theme

Examples

data("twoCmt")
distMult(twoCmt)
distMult(twoCmt,order = c("Boxplot","Density","QQ"))
