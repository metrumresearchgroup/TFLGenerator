OBSvIPREDR Documentation

Standard goodness of fit plots for observed versus predicted values

Description

Yields a ggplot object comparing observed values to predicted values

Usage

OBSvIPRED(datFile,  xBy = "IPRE", yBy = "DV", groupBy=NULL, markBy = NULL, xLimit = NULL, yLimit = xLimit, xBreaks = lseq(datFile[c(xBy, yBy)]), yBreaks = xBreaks, xForm = waiver(), yForm = xForm, xScale = log10_trans(), yScale = xScale, Title = "", xLab = "Individiual Prediction", yLab = "Observed Plasma Concentration", facetBy = "", minorTicks = FALSE, ...)

Arguments

datFile

data.frame output of a NM run containing ER-SSAP standard column names and numeric-only columns

xBy

character column name for x-axis

yBy

character column name for y-axis

groupBy

character column name for groupings within the plot (such as smoothing line)

markBy

character column name for characteristic groupings within the plot (such as color and shape)

xLimit

vector input for x-axis limits example: c(.1,100000)

xBreaks

vector input for x-axis breaks. Best set as a log sequence of the range of x and y data.

xForm

function format of the x-axis (options: comma, scientific, percent or waiver() for no formatting)

xScale

function format for the x-axis (options, "identity", log10_trans(), log_trans())

Title

character figure title

xLab

character x-axis title

yLab

character y-axis title

facetBy

character column name for optional faceting

minorTicks

logical indicating whether minor log 10 ticks should be added to the y and x axis

yLimit

vector input for y-axis limits example: c(.1,100000). For best comparison should be left as the default (set to xLimit)

yBreaks

vector input for x-axis breaks. For best comparison should be left as the default (set to xBreaks)

yForm

function format of the y-axis (options: comma, scientific, percent or waiver() for no formatting)

yScale

function format for the y-axis (options, "identity", log10_trans(), log_trans())

Details

Standard goodness of fit showing DV versus population prediction or individual prediction.

Value

returns ggplot object (grob) that must be saved or printed

Examples

data("twoCmt")
OBSvIPRED(twoCmt%>%filter(DV>0),xBy='IPRED')
OBSvIPRED(twoCmt,xBy='IPRED',xScale = 'identity',markBy = 'SEX')
