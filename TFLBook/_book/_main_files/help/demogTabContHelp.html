demogTabContR Documentation

Demographics table for continuous variables

Usage

demogTabCont(datFile, groupBy = list(c("STUDY", "Study")), idVar = "NMID",
  conList = list(c("AGE", "Age (years)"), c("HGTB", "Height (cm)"), c("WGTB",
  "Weight (kg)")), catList = list(c("DIS", "Disease")), sigDig = 3,
  srcAdd = TRUE, srcPath = ".", srcName = "script",
  tblPath = "../deliv/table", tblName = "table.tex")

Arguments

datFile

data.frame

groupBy

a list of (multiple) two element vectors; the first element is the column of datFile on which the table is grouped and the second element is the "pretty" name for printing

idVar

ID variable in datFile

conList

a list of (multiple) two element vectors (like groupBy) that denote the list of continuous variables you wish to summarize

catList

a list of a single two element vector giving the variable on which to stratify by. Pass an empty string for no stratification

sigDig

number of significant digits

Value

LaTex table

See Also

texPreview

 Knit<-!is.null(knitr::opts_knit$get('rmarkdown.pandoc.to')) #Check if in knit environment
