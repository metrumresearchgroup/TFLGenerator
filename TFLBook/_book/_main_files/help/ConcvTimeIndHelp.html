ConcvTimeIndR Documentation

Concentration vs time profiles for groups of patients (spaghetti plot)

Description

Longitundinal plot showing individual concentration vs time profiles, including observed, conditional, and unconditional (on random effects) values of concentration

Usage

ConcvTimeInd(datFile, groupBy = "NMID", xBy = "TAFD", yBy = "DV",
  markBy = "DOSE", markByType = "Discrete", preserveMarkByLevels = F,
  Color = T, xLimit = NULL, yLimit = NULL, xForm = waiver(),
  yForm = waiver(), minorTicks = FALSE, minorTickNum = 10,
  xScale = "identity", yScale = "log10", Title = "Individuals",
  xLab = "Time", yLab = "Concentration", fF = "", facetByRow = ".",
  facetByCol = ".", fnrow = NULL, fncol = NULL, fscales = "fixed",
  facetType = "wrap", themeUpdate = list(), themeTextSize = 14,
  themePlotTitleSize = 1.2, themeAxisTxtSize = 0.8,
  themeAxisTxtColour = "black", themeAxisTitleTxtSize = 0.9,
  themeAxisTitleColour = "black", themePanelBackgroundFill = "white",
  themePanelGridSize = NULL, themePanelGridColour = "white",
  themePanelLineType = 1, themePanelTitleSize = 1.2,
  themePlotTitleColour = "black", themePlotLegendPosition = "right",
  srcAdd = TRUE, srcPath = ".", srcName = "script",
  figPath = "../deliv/figure", figName = "Rplot.pdf", ...)

Arguments

datFile

A data.frame containing the necessary variables for the figure

groupBy

character column name for groupings within the plot (such as smoothing line)

xBy

Name of the variable denoting the x axis (typically time) in datFile

yBy

Name of the variable containing the observed concentration values

markBy

character column name for characteristic groupings within the plot (such as color and shape)

markByType

character, controls the type of variable expected mapped to markBy (Discrete, Continuous)

preserveMarkByLevels

Logical, to preserve the levels of the markBy variable

Color

Logical, controls if the plot has color aesthetic.

xLimit

Two element numeric vector giving the lower and upper limits of xBy

yLimit

Two element numeric vector giving the lower and upper limits of the concentration axis (y)

xForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the x-axis

yForm

One of "percent", "comma", "none", or "scientific" denoting the labeling to be used for the y-axis

minorTicks

Logical, include minor ticks?

minorTickNum

Integer, number of minor ticks between major ticks

Title

String, the title for the plot

theme*

Various controls for the ggplot2 theme

Value

A concentration vs time ggplot2 object of class TFL which can be plotted or printed with the corresponding method.

Examples

data("twoCmt")
twoCmt=twoCmt%>%mutate(DOSE=factor(DOSE))
ConcvTimeInd(datFile = twoCmt)
ConcvTimeInd(datFile = twoCmt,facetByRow = 'SEX')
