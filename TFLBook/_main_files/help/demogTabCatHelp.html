demogTabCatR Documentation

Demographics table for categorical variables

Usage

demogTabCat(datFile, groupBy = list(c("STUDY", "Study")), idVar = "NMID",
  catList = list(c("SEX", "Sex"), c("RACE", "Race")),
  stratBy = list(c("DIS", "Disease")), sigDig = 3)

Arguments

datFile

data.frame

groupBy

a list of (multiple) two element vectors; the first element is the column of datFile on which the table is grouped and the second element is the "pretty" name for printing

idVar

ID variable in datFile

catList

a list of (multiple) two element vectors (like groupBy) that denote the list of categorical variables you wish to summarize

stratBy

a list of a single two element vector giving the variable on which to stratify by. Pass an empty string for no stratification

sigDig

number of significant digits

Value

LaTex table

Examples

data("twoCmt")
ex1=demogTabCat(twoCmt)
ex2=demogTabCat(twoCmt,stratBy = '')
list(ex1,ex2)
